# Java 简介

## 概述

Java 是一种面向对象、跨平台的编程语言，由 Sun Microsystems（现属 Oracle）于 1995 年发布。其核心理念是 **"一次编写，到处运行"（Write Once, Run Anywhere）**，通过 Java 虚拟机（JVM）实现跨平台能力。

## 核心特性

- **跨平台**：编译后的字节码运行在 JVM 上，不依赖具体操作系统
- **面向对象**：支持封装、继承、多态等面向对象编程范式
- **强类型语言**：编译时进行严格的类型检查，减少运行时错误
- **自动垃圾回收**：内置 GC 机制管理内存，降低内存泄漏风险
- **丰富的生态**：拥有庞大的标准库和第三方框架（如 Spring、Hibernate）

## 主要应用领域

| 领域 | 典型应用 |
|------|---------|
| 企业级后端 | Spring Boot、微服务架构 |
| 安卓开发 | Android SDK |
| 大数据 | Hadoop、Spark、Kafka |
| 金融系统 | 银行交易、支付平台 |

## 简单示例

```java
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

## 总结

Java 凭借稳定性、高性能和成熟的生态系统，至今仍是全球使用最广泛的编程语言之一，尤其在企业级应用开发中占据重要地位。
